Message Classification

You can train / use per domain a message classifier, whereas the following REST interfaces are available:

  • Import dataset to train classifier (Sample dataset)
  • Get all classifications / labels
  • Export dataset
  • (Re-)Train classifier
  • Classify a message
  • Rate classified message
  • Get Classifications Preference Dataset
  • Run classification benchmark

TOPdesk Integration

The classifier can be trained with existing TOPdesk incidents and the labels can be synchronized continuously, whereas in order to do so, please make sure to have TOPdesk configured as Knowledge Source.

  • Import batch of incidents
  • Sync categories / subcategories
