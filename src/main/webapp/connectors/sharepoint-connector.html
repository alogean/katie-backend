SharePoint Connector

You can connect SharePoint with Katie using the following REST interface: https://app.katie.qa/swagger-ui/#/domain-controller/addKnowledgeSourceSharePointUsingPOST

Requirements

In order to connect SharePoint with Katie you need to provide the following information / parameters:
  • SharePoint Base URL, e.g. https://wyona.sharepoint.com/sites/KatieTest
  • SharePoint Site Id, whereas get Id for example by using Microsoft Graph: https://graph.microsoft.com/v1.0/sites?search=KatieTest
  • Microsoft Azure Tenant Id. whereas see https://portal.azure.com/#settings/directory
  • Microsoft Azure App / Client Id and Secret, whereas see Microsoft Entra Id / Applications
  • Configure permissions for Microsoft Azure App / Client, e.g. give consent for Sites.Read.All

Limitations

SharePoint pages of Content Type "Wiki Page" do not get loaded!

Also see https://learn.microsoft.com/en-us/answers/questions/629474/get-wiki-page-content-from-sharepoint-using-graph
