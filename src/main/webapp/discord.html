How does Katie work inside a Discord channel?

  • A team member asks a question inside a Discord channel, for example "What is the mobile phone number of Katie?" or "Who can recommend a good neuropsychologist?"
  • Katie will analyze the question, for example check whether it is a duplicated question and might has been answered before
  • If Katie is able to provide a good answer, then this was it already :-) otherwise please continue reading ...
  • If Katie does not know the answer or if Katie's answer was not helpful or not correct ...
    • ... then with a click of a button Katie can forward the question to an expert affiliated with the Discord server/channel
    • ... and when the expert or a teammate is answering the question or correcting Katie's answer, then Katie will learn from the expert's or the teammate's answer and will be able to answer the same or a similar question next time.

Consider Katie as a human being who is eager to learn, but with a much better memory and never being impatient :-)

Configure your own Katie Discord Bot and connect a Discord channel with a Katie domain

  • https://discord.com/developers/applications
  • Click on "New Application" and enter a Name and click "Create"
  • Add an App icon and a short Description
  • Click in the right menu on "Bot" and click on "Reset Token" and copy the generated token and very IMPORTANT, enable "Message Content Intent", because otherwise no messages will be sent to Katie.
  • Enable Discord (set application property discord.enabled or DISCORD_ENABLED to true)
  • Add the token to Katie (set application property discord.bot.token (inside secret-keys.properties) or DISCORD_BOT_TOKEN), such that Katie can register with Discord
  • Add bot to your Discord server, e.g. https://discord.com/oauth2/authorize?client_id=1196487754297643058&scope=bot&permissions=277025508352
  • Enter the command /invite in one of your Discord server channels and Katie will ask whether you want to connect this channel with a Katie domain
  • Connect the Discord channel with a Katie domain using /swagger-ui/#/discord-bot-controller/connectGuildChannelWithDomainUsingGET, whereas generate a JWT token using /swagger-ui/#/authentication-controller/generateGenericJWTUsingPOST
