How does Katie work with TOPdesk?

  • A customer asks a question, for example "I have a new mobile phone number, how can I reset 2 factor authentication?"
  • Katie will analyze the question, for example check whether it is a duplicated question and might has been answered before
  • If Katie is able to provide a good answer, then the customer care agent can forward Katie's answer to the customer or otherwise replace / correct Katie's answer and Katie will learn accordingly.

Consider Katie as a human being who is eager to learn, but with a much better memory and never being impatient :-)

Integration Katie with TOPdesk?

In TOPdesk you can define Events (the when) that trigger a certain Action (the what).

The most simple integration is to define an "Aktionsabfolge" with two actions:

  • Action 1: Forward question to Katie using the REST Interface of Katie
  • Action 2: Writing the answer of Katie into TOPdesk
